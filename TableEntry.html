API: Key Frame

Our module facilitates the animation creation process. Our module consist of key framer and animation player.

Key Framer:

Description: is a map comprising renderables as keys and linked array of different animations as values.
Basic functions: add, or delete frame animation, create new animation associate with the renderable, add/delete/get a renderable to the map.
Example:
  • A renderable for first box can be green with animation changing its color to white while moving left
  • You then can add a different animation for it changing its color to red while moving right
  • You can then add another renderable for second box that is rotating to the left.

Animation player

Description is responsible for playing the current animation of the object Basic functions play the animation default at linear interpolation, unless user provide their own code
